Adımlar:

1)  Herhangi bir IDE'de (Pycharm, vb.) yeni bir Python projesi oluşturun.
2)  Komut satırını (cmd) çalıştırın ve komut satırında açmış olduğunuz Python projenizin dizinine gidin.
3)  Komut satırına aşağıdakini yazarak gerekli olan kütüphaneyi projenize yükleyin.
	pip install IMDbPY
4)  GitHub üzerinden erişebildiğiniz main.py dosyasını bilgisayarınıza indirin ve proje dizininin içerisine taşıyın.
5)  main.py dosyasını çalıştırın.
